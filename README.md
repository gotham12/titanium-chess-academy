# Titanium Chess Academy

Premium marketing website for [Titanium Chess Academy](https://www.titaniumchessacademy.com) — year-round chess lessons and Summer Camp 2026 in Shrewsbury, MA.

## Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lenis smooth scroll

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deploy to [Vercel](https://vercel.com) or any Next.js-compatible host:

```bash
npm run build
npm start
```

## Pages

- `/` — Home
- `/summer-camp` — Summer Camp 2026 details
- `/register` — Enrollment form
